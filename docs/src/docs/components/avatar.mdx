## Avatar

```js
import { Avatar } from "athenic-ui";
```

### Example

```jsx
<Avatar size="small" link={url} notifications={4} />
```

### Props

- image?: `string`
- className?: `string`
- style?: `CSSProperties`
- size?: `AvatarSize`
- notifications?: `number`
- link?: `string`
- onClick?: `() => void`

## Types

- AvatarSize: `"small" | "medium" | "large"`
