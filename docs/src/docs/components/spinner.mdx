## Spinner

```js
import { Spinner } from "athenic-ui";
```

### Examples

```jsx
<Spinner variant="secondary" />
<Spinner size="small" />
<Spinner speed={500} />

```

### Props

- variant?: `SpinnerVariant`
- size?: `SpinnerSize`
- speed?: `SpinnerSpeed`
- style?: `CSSProperties`

## Types

- SpinnerVariant: `"default" | "primary" | "secondary" | "tertiary"`
- SpinnerSize: `"small" | "medium" | "large"`
- SpinnerSpeed: `number`
