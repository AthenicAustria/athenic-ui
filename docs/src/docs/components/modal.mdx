### Props

- shown: `boolean`
- title?: `string`
- closeButton?: `boolean`
- className?: `string`
- style?: `CSSProperties`
- onClose: `() => void`
- children?:` ReactChild | ReactChildren | ReactChild[] | ReactChildren[]`
- position?: `ModalPosition`
- closeOnExternalClick?: `boolean`

## Types

- ModalPosition: `"default" | "center" | "bottomLeft" | "bottomRight"`
