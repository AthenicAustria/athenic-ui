## Rating

```js
import { Rating } from "athenic-ui";
```

### Examples

```jsx
<Rating variant="star" size="small" />
<Rating variant="star" />
<Rating variant="star" size="large" />
<Rating variant="star" initialRating={3} />
<Rating variant="emoji"/>

```

### Props

- variant?: `RatingVariant`
- size?: `RatingSize`
- className?: `string`
- style?: `CSSProperties`
- initialRating?: `number`
- onRate?: `(value: number) => void`

## Types

- RatingVariant: `"star" | "emoji"`
- RatingSize: `"small" | "medium" | "large"`
- EmojiType: `{ emoji: string; value: number }`
- EmojiScale: `EmojiType[]`
