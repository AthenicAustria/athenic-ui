## Radio-Group

```js
import { SearchInput } from "athenic-ui";
```

### Examples

```jsx
<SearchInput />
<SearchInput disabled />
<SearchInput validationMessage="Please enter a valid search term!" />
<SearchInput
  onSearch={() =>
    window.alert(
      "You must be looking for something very important huh?"
    )
  }
/>
<SearchInput expandOnFocus />
```

### Props

- style?: `CSSProperties`
- className?: `string`
- label?: `string`
- name?: `string`
- form?: `string`
- disabled?: `boolean`
- validationMessage?: `string`
- invalid?: `boolean`
- onChange?: `(e: React.FormEvent<HTMLInputElement>) => void`
- onFocus?: `(e: React.FormEvent<HTMLInputElement>) => void`
