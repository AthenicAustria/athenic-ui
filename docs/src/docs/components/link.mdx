## Link

```js
import { Link } from "athenic-ui";
```

### Example

```jsx
<Link to={`https://github.com/AthenicAustria/athenic-ui`} text="Click me!" />
```

### Props

- text: `string`
- to: `string`
- target?: `string`
- className?: `string`
- style?: `CSSProperties`
